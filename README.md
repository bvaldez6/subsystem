## Setup
### back end
- install `uv`
- `cd backend`
- `uv venv` to create a venv
- activate the venv:
  - macos / linux: `source .venv/bin/activate`
  - windows: `.venv\Scripts\Activate.ps1` or `.venv\Scripts\activate.bat`
- run `uv sync` to install project's python packages
